# Aramis Shop
This project is a store that is powered by Tilwind CSS and has a bit of JavaScript and is fully responsive on all devices
# Installation 
Npm run Tailwind 

# Technologies
-HTML, 
-Tailwind
-JavaScript
#   Features
- responsive for all devices
- Sales of various products
- Cleancode
